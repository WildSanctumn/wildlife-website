# PLMun Wildlife Conservation Hub

## Project Overview

It shows information about endangered animals in the Philippines. The goal is to help people learn more about wildlife conservation and why it is important to protect animals.

---

## Project Goals

- To teach people about endangered animals in the Philippines  
- To raise awareness about wildlife conservation  
- To practice basic web development using HTML and CSS  
- To create a simple and informative website  

---

## Tools Used

- HTML – for the structure of the website  
- CSS – for design and styling  
- Bootstrap – for layout and forms  
- Unsplash – for background images  
- GitHub Pages – for hosting the website  

---

## Pages in the Website

- Home – introduction and list of animals  
- Gallery – collection of animal images  
- Contact – contact form for messages  
- About Us – information about the project and members  

---

## How to Run the Website

Option 1 (Offline):
1. Download or extract the project folder  
2. Open `index.html`  
3. Run it using Google Chrome or any browser  

Option 2 (Online):
- Open the GitHub Pages link of the website
