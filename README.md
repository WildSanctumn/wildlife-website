# 🌿 PLMun Wildlife Conservation Hub

## 📌 Project Overview

The **PLMun Wildlife Conservation Hub** is a simple school project website that focuses on raising awareness about endangered wildlife in the Philippines.  

The website shows different endangered animals, their scientific names, and basic information about why they need to be protected.

This project was created as part of our academic requirement in college to practice building a website using HTML and CSS.

---

## 🎯 Project Goals

The main goals of this project are:

- To help people learn about endangered animals in the Philippines  
- To promote awareness about wildlife conservation  
- To practice basic web development skills  
- To create a simple and informative website for students  

---

## 💻 Technologies Used

This project was built using the following technologies:

- HTML5 – for the structure of the website  
- CSS3 – for design and styling  
- Bootstrap 5 – for responsive layout and form design  
- Unsplash – for background images  
- GitHub Pages – for hosting the website  

---

## 📁 Website Pages

This project includes the following pages:

- **Home Page** – Introduction and list of endangered animals  
- **Gallery Page** – Collection of wildlife images  
- **Contact Page** – Simple contact form for messages  
- **About Us Page** – Information about the project and members  

---

## ▶️ How to Run the Project Locally

If you want to run this project on your computer, follow these steps:

1. Download or clone the repository:

2. Open the project folder

3. Find the file named: index.html
   
4. Open it using any web browser (Google Chrome recommended)

5. The website will run locally without needing internet (except images if online links are used)

---

## 🌐 How to Deploy on GitHub Pages

If you want to publish the website online:

1. Go to your GitHub repository  
2. Click **Settings**  
3. Go to **Pages** section  
4. Under “Source”, select:
- Branch: `main`
- Folder: `/root`  
5. Click **Save**  
6. Wait a few minutes, then your site will be live
